# Budget_Tracker
## Usage Guide
This application allows you to track your income and expenses while visualizing your budget with interactive charts.
1. Default View
* When you first open the app, the tables and charts display default placeholder values.
* These values update automatically as you add your own income or expense data.
2. Adding Income or Expenses
* Click the Add Income or Add Expenses button to reveal the input form.
* The corresponding details dropdown will also automatically open, showing the table for that category.
* Fill in the following fields:
    * Category: Select a predefined category from the dropdown.
    * Custom Category: If you choose "Custom," a field will appear allowing you to enter your own category name.
    * Amount: Enter the numerical value for the income or expense.
    * Notes (Optional): Add any additional details if needed.
* After entering the details, click Add.
* The new entry will be added to the table, totals will update, and the charts will reflect the new data.
3. Viewing Data
* Both Income and Expenses sections have a details dropdown where you can see all added entries.
* Each table displays:
    * Category
    * Amount
    * Notes
4. Charts
* Bar Chart: 
    * Compares total income vs total expenses.
    * Informs you whether or not goal is met.
* Pie Chart: 
    *  Shows a 50/30/20 budget breakdown (Needs / Wants / Savings).
    * Breaksdown Income value based on budget.
